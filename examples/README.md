# Example Posts

This directory contains example markdown files demonstrating md2wp features.

## Examples

- **basic-post.md** - Basic post with headings, lists, code, and blockquotes
- **post-with-images.md** - Post demonstrating image handling (requires sample images)
- **minimal-post.md** - Minimal post with just required frontmatter

## Testing

Test dry-run mode with any example:

```bash
pnpm cli publish examples/basic-post.md --dry-run
```

## Images

Sample images provided by [Cat as a Service (cataas.com)](https://cataas.com).

You can add more sample images to `examples/images/` to test image upload functionality.
